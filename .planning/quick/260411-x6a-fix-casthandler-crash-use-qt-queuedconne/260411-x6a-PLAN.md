<objective>
Fix heap corruption crash caused by CastSession being destroyed while still inside its own onDisconnected slot.

Purpose: CastHandler::onSessionFinished calls m_session.reset() which destroys the CastSession. When connected with AutoConnection (direct, same thread), this destruction happens while CastSession::onDisconnected is still on the call stack, causing malloc heap corruption. Using Qt::QueuedConnection defers the slot invocation to the next event loop iteration, ensuring the signal emitter has fully returned before destruction.

Output: Patched CastHandler.cpp with Qt::QueuedConnection on both finished signal connections.
</objective>
